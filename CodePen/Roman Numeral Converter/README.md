# Roman Numeral Converter

## Project Description
This is a simple web application that converts a given number (between 1 and 3999) into its Roman numeral equivalent. It validates the input to ensure it is a number within the valid range, and if the input is invalid, it provides an error message.

## Features
- Converts numbers between 1 and 3999 to Roman numerals.
- Provides validation for input to ensure it is a valid number within the range.
- Displays an error message for invalid inputs.
- Interactive user interface with a responsive design.

## Technologies Used
- **HTML**: For the structure of the webpage.
- **CSS**: For styling the user interface.
- **JavaScript**: For handling the conversion logic and user interactions.

## How to Use
1. Enter a number between 1 and 3999 into the input field.
2. Click the **Convert** button to convert the number into its Roman numeral equivalent.
3. The Roman numeral will be displayed below the input field. If the input is invalid, an error message will appear instead.
